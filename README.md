### 说明!

<p>本服务端基于国服【潜能觉醒】客户端开发</p>
<a href="http://download.online-game.com.cn/client/hyonline2021.exe">[下载地址]</a>

### 运行服务端程序

<p>1.自行安装mysql 将hyonline.sql导入</p>
<p>2.修改配置文件中sql连接信息.</p>
<p>3.启动springboot</p>


### 运行客户端程序

<p>找到客户端HyData文件夹，修改server.ini中的服务器地址为127.0.0.1,然后进入游戏</p>
<p>默认账号hy00001</p>
<p>默认密码123456</p>
